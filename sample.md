# Hello Markdown

## Introduction

This is a **markdown reader** built with Ink and marked.

## Code Example

```javascript
const greet = (name) => `Hello, ${name}!`;
console.log(greet('world'));
```

## Features

- Headings with color
- Paragraphs
- Code blocks
- Lists

## Keybindings

| Key | Action |
|:----|-------:|
| `j` / `↓` | Scroll down |
| `k` / `↑` | Scroll up |
| `g` / `G` | Top / Bottom |
| `q` | Quit |
